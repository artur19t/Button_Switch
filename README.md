# Базовый проект: Переключение светодиода по нажатию кнопки

Микроконтроллер: **STM32F103C8**  
Среда: **CMSIS**, без использования HAL, LL и прерываний.

## Описание проекта
Данный пример демонстрирует базовую работу с GPIO на STM32F103: при нажатии кнопки происходит включение светодиода, при отпускании - выключение. Проект выполнен на чистом CMSIS — работа с регистрами осуществляется напрямую.

## Аппаратная конфигурация
- Светодиод подключён к **PB2** (типичная плата Blue Pill+).
- Кнопка подключена к **PA0** и настроена как вход с подтяжкой.
- Питание: 3.3 В.

## Логика работы
1. Микроконтроллер настраивает тактирование порта GPIOA и GPIOC.
2. PB2 конфигурируется как выход push‑pull на низкой скорости.
3. PA0 конфигурируется как вход с подтяжкой.
4. В основном цикле программа опрашивает состояние кнопки.
5. При обнаружении нажатия/отпускания — изменяет уровень на пине PC13.


## Возможные улучшения
- Добавить прерывание по кнопке.
- Добавить программный или аппаратный антидребезг.
- Перевести конфигурацию в отдельные модули `gpio.c / gpio.h`.
